# Day 001: Lógica Básica e Condicionais

Implementar uma função em JavaScript chamada `classificarNumero(numero)` que classifica um número inteiro como:

1.  "Neutro" (se for zero)
2.  "Negativo" (se for menor que zero)
3.  "Positivo e Par"
4.  "Positivo e Ímpar"
